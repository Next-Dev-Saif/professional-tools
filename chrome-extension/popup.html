Hephaestus Bridge
Relays agent commands to localhost:3000
Checking...
External AI agents can use this bridge to automatically fill out forms and generate content on the Hephaestus web app.
How to instruct your AI Agent:
  1. Connect: Tell the agent: "Send a POST request to http://localhost:3000/api/agent/relay with your commands."
  2. Command Format: The payload must be:
    {"action": "fill_fields", "fields": {"fieldName": "value"}}
  3. Execution: The agent should inspect the web page for field names, formulate the JSON payload, and POST it. This extension will immediately inject the data.
